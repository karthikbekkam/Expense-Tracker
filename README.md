Expense Tracker

Description:
A simple Expense Tracker application that allows users to add, view, and manage daily expenses. The project helps in tracking spending by recording expense title, amount, and date, and displays the total expenses. This project is suitable as a university mini project.

Technologies Used:
HTML
CSS
JavaScript

Features:
Add expense with title, amount, and date
View list of expenses
Calculate total expense amount
Simple and responsive user interface

How to Run:
Open the index.html file in any web browser.

Project Use:
Practice of frontend development concepts
